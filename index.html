安装说明
⚙️ 技术栈
Python 3.14 pywebview 6.x Quartz.CGEventPost AppKit.NSApplication WKWebView Obsidian Markdown
fn 键通过 CGEventPost 硬件级注入，触发 Typeless 语音输入
WKWebView 渲染 UI，无边框透明浮窗
进程忽略 SIGHUP，Terminal 关闭后持续运行
📦 安装步骤
1
克隆仓库到本地
git clone https://github.com/yehloolau-afk/ai-daily-note.git
2
创建虚拟环境并安装依赖
python3 -m venv venv
venv/bin/pip install pywebview pyobjc
3
双击 启动记录器.command 启动工具
4
首次运行需在「系统设置 → 隐私 → 辅助功能」授权 Terminal
📌 注意事项
前提：需已安装 Typeless，并在 Typeless 中将 fn 键设为语音输入快捷键
权限：CGEventPost 注入按键需要「辅助功能」权限，首次启动时系统会引导授权
保存位置：默认写入 ~/Documents/.../AI收获记录.md，在 recorder.py 顶部修改 OBSIDIAN_FILE 路径
macOS 专属：依赖 AppKit / Quartz，仅支持 macOS 12+
